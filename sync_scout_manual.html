はじめに

Sync Scout とは

Sync Scout は、バレーボールの試合分析データ（DVW ファイル）と YouTube 動画を連携させ、特定のプレーシーン（サーブ・レセプション・アタックなど）を素早く検索・再生できる Web ビューアです。

試合のスコアデータと映像を紐づけることで、「あの選手のアタックシーンだけを連続再生したい」「エラーになったサーブを見直したい」といった要望に即座に応えられます。

動作環境

Sync Scout は Web ブラウザ上で動作します。PC・スマートフォン・タブレットのいずれからもアクセスできます。

推奨ブラウザ（PC）

Google Chrome（最新版）
Microsoft Edge（最新版）

推奨ブラウザ（スマホ）

Safari（iOS）
Chrome（Android）

対応ファイル形式

.dvw DataVolley 形式のスコアファイル

動画ソース

YouTube（公開・限定公開）

スマートフォンでは DVW ファイルのアップロードに対応していないため、試合の登録は PC から行う必要があります。登録済みの試合の閲覧・再生・絞り込み・共有はスマートフォンでも利用できます。

YouTube の動画は事前にアップロードされている必要があります。未アップロードの動画は使用できません。

第1章

ログイン・ログアウト

💻 PC 📱 スマートフォン

ログイン

Sync Scout を使用するには、合言葉(パスコード)でのログインが必要です。PC・スマートフォンどちらからでもログインできます。

1

ブラウザで Sync Scout の URL にアクセスします。

2

ログイン画面が表示されたら、チームに割り当てられた合言葉(パスコード)を入力します。

3

ログインボタンを押すとメイン画面に移行します。

スマートフォンでのログイン
⌨️
入力欄をタップ
合言葉(パスコード)欄をタップするとキーボードが表示されます
🔐
パスワード自動入力
iOS ではキーチェーン、Android ではパスワードマネージャーからの自動入力が利用できます

ログアウト

画面右上の Logout ボタンをクリック（タップ）するとログアウトできます。スマートフォンでは、メニューを開いてからログアウトボタンにアクセスします。

共有端末を使用している場合は、作業終了後に必ずログアウトしてください。スマートフォンのブラウザはログイン状態が保持されやすいため、特にご注意ください。

第2章

試合の登録・管理

💻 PC（登録・削除） 📱 スマートフォン（閲覧のみ）

試合の登録・削除は PC からのみ行えます。DVW ファイルのアップロードにはファイルシステムへのアクセスが必要なため、スマートフォンからは実行できません。登録済みの試合の選択・閲覧はスマートフォンでも可能です。

試合を新しく追加する（PC）

分析したい試合を登録します。YouTube の動画 URL と対応する DVW ファイルをセットで登録することで、映像とスコアデータが連携されます。

1

画面左上の ➕ Add Match ボタンをクリックします。

2

Category（カテゴリ）を入力します（例：SVリーグ、大学リーグ、練習など）。

3

YouTube URL フィールドに動画の URL を貼り付けます。

4

DVW File フィールドで .dvw ファイルを選択してアップロードします。

5

Upload & Save ボタンをクリックして登録完了です。

DVW ファイルは DataVolley 形式（.dvw）のみ対応しています。VolleyStation 形式（.vsm）は現在非対応です。

試合を選択する

登録済みの試合は Select Match... のドロップダウンから選択できます。PC ではクリック、スマートフォンではタップで操作します。

スマートフォンでの試合選択
👆
ドロップダウンをタップ
Select Match... をタップするとリストが表示されます
↕️
スクロールして選択
試合数が多い場合はリストをスクロールして目的の試合を見つけます

試合を削除する（PC のみ）

削除したい試合を選択した状態で、🗑️ アイコンをクリックします。確認ダイアログが表示されますので、内容を確認してから削除してください。

削除した試合データは元に戻せません。操作前に必ず内容を確認してください。

表示モード・プレーの絞り込み

💻 PC 📱 スマートフォン

プレーリストの表示形式を 4 種類のモードで切り替えられます。スマートフォンでも全モードに対応しており、モード名をタップして切り替えます。

Rallies

Individual
選手別の個人データ表示。特定選手のパフォーマンス分析に適しています。

ローテーション別の表示。ローテーションごとの得失点傾向を把握できます。

各モードの説明

Ralliesモード

DVW ラリー単位で一覧表示するモードです。プレー間やセット間、タイムアウトを省いて動画を見られるため、試合全体の流れを把握したいときに便利です。

表示するサーブチームを選びます。
Both Teams

両チームのサーブを表示します。試合全体を見る場合に使用します。

home Team

ホームチームのサーブを表示します。自チームの場合、ブレイクポイントだけ確認したい場合に使用します。

away Team

アウェイチームのサーブを表示します。自チームの場合、サイドアウトだけ連続で確認したい場合に使用します。

Individualモード

選手別の個人データ表示モードです。特定選手のパフォーマンス分析に適しています。

Tableモード

Table タブを選択すると、表形式でサーブ・レセプション・アタックのスキルの成績が一覧表示されます。各行にはチーム・選手・スキル・評価などの情報が表示されます。青い数字をクリック（スマートフォンではタップ）することで該当プレーの映像を再生できます。直感的にプレーを選択したい場合に便利なモードです。

Rotationモード

Rotation タブを選択すると、両チームのローテーションごとにサイドアウトフェイズ、ブレイクポイントフェイズが表示されます。サイドアウト65%以上、ブレイク40%以上の数字は赤字で表示されます。青い数字をクリック(タップ)すると映像にジャンプできます。
ローテーションごとにプレーを確認したいときや、対戦相手の対策に使用することができます。

コメント・タグで検索

🔍 Search Comments / Tags の検索欄に文字を入力すると、コメントやタグが付与されたラリーを絞り込めます。スマートフォンでは入力欄をタップしてキーボードから入力します。

Auto-Skip（オートスキップ）

Auto-Skip を ON にすると、絞り込み条件に合致したラリーを再生後、自動的に次のラリーへスキップします。手動でボタンを押さずに連続確認できるため、大量のプレー確認に便利です。スマートフォンでもボタンのタップで ON / OFF を切り替えられます。

第7章

共有機能

💻 PC 📱 スマートフォン

現在の絞り込み・表示状態を他のユーザーと共有できます。スマートフォンからも LINE や各種メッセージアプリへ直接共有できます。

URL をコピー

Copy URL ボタンで現在の状態の URL をクリップボードにコピーします。メールやSNSで送るとそのまま同じ画面を共有できます。スマートフォンでもタップでコピーできます。

LINE で共有

Share LINE ボタンで LINE のシェア画面が開き、トークやグループに送信できます。スマートフォンからの操作が特に便利です。

送るメッセージには合言葉が含まれます。チーム内でまだSync Scoutを使用していない人でも、動画を見ることができますが、広く公開したり、チーム以外のユーザーと共有することにはご注意ください。

スマートフォンでの共有方法
🔗
Copy URL をタップ
URL がコピーされます。SMS・メール・各種 SNS に貼り付けて共有
💚
Share LINE をタップ
LINE アプリが起動し、送信先のトーク・グループを選んで送信
📤
ブラウザの共有機能
iOS Safari の「共有」ボタンや Android Chrome の「共有」からも URL を送れます
付録

よくある質問（FAQ）

DVW ファイルをアップロードしたが、プレーデータが表示されない ▼
DVW ファイルが正しい DataVolley 形式かどうかご確認ください。VolleyStation 形式（.vsm）はサポートされていません。また、ファイルが破損していないか、DataVolley アプリで開けるかどうかも確認してみてください。
YouTube 動画が表示されない・再生できない ▼
以下を確認してください。① YouTube URL が正しく入力されているか。② 動画が「公開」または「限定公開」設定になっているか（非公開は不可）。③ ブラウザの Cookie / JavaScript が有効になっているか。スマートフォンの場合、Safari の「コンテンツブロッカー」や Chrome の設定が影響することがあります。
動画の再生タイミングとプレーデータがずれる ▼
DVW ファイルに記録されたタイムスタンプと YouTube 動画の開始タイミングにずれがある可能性があります。-3s / +3s ボタンや ← / → キー（PC）で手動調整してください。スマートフォンでは -3s / +3s ボタンをタップして調整します
またアップロードした動画が、スカウトファイルと同期した動画ファイルと同じものか確認してください。。
キーボードショートカットが反応しない ▼
テキスト入力欄（コメント欄など）にフォーカスが当たっているとショートカットが無効になります。入力欄以外の場所をクリックしてフォーカスを外してから再度お試しください。なお、キーボードショートカットは PC 専用の機能です。スマートフォンでは画面上のボタンをご利用ください。
スマートフォンで試合を登録しようとしたが Add Match ボタンが見当たらない ▼
試合の登録（DVW ファイルのアップロード）はスマートフォンからは行えません。PC のブラウザからログインして試合を登録してください。登録済みの試合の閲覧・フィルタリング・共有はスマートフォンでも全て利用できます。
